预约电子会议室的订单 API

用户创建新的预约订单成功

POST /orders

Parameters

Name Description
order[appoint_name] 预约人姓名
order[appoint_phone] 预约人联系电话
order[join_number] 参加会议的人数
order[appoint_at] 预约会议日期
order[session] 会议室开放时间
order[state] 预约订单的状态
order[room_layout] 预约会议室的布局
order[is_projector] 预约是否需要投影仪
order[is_meeting_phone] 预约是否需要会议电话
order[meeting_room_id] 预约的会议室
order[sign] 预约的会议室
